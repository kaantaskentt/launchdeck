# Roadmap

## Now

- Local folder, zip, and public GitHub imports.
- Project health signals from package metadata and repository files.
- Safe static previews with scripts blocked by default.
- Local editor and file updates.
- Desktop and mobile responsive cockpit UI.

## Next

- Framework-specific preview adapters for Vite, Next.js, Astro, and static sites.
- Safer per-project trust settings for script-enabled preview.
- More complete command palette actions.
- Project search, tags, and pinned workspaces.
- Import fixtures for common repo shapes.
- Optional File System Access API write-back.

## Later

- Desktop companion for secure local process execution.
- Git status integration.
- GitHub issue/PR context panel.
- Extension/plugin model for framework analyzers.
- Shareable read-only project snapshots.
